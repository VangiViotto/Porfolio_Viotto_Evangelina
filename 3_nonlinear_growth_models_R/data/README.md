# Data

This folder contains the datasets used in the analysis.

## Files

- `CIPromediosadultosPyWyPYprom.csv`  
- `CIadultos_masked.csv`  

## Note

The original datasets cannot be shared due to confidentiality restrictions.

- Some datasets were fully simulated to reproduce the structure and distribution of the original data  
- Others were derived from real data with added noise (masked) to preserve statistical properties  

Although these datasets appear independent, they are conceptually linked.

Size-frequency analysis (Script 1) was used to evaluate the size structure of early life stages (hatchlings), which informed the interpretation and parameterization of the growth models implemented in Script 2.

Due to data confidentiality constraints, this relationship is not explicitly represented in the shared datasets, and they are provided here as separate files.

These datasets are intended for reproducibility of the analytical workflows only.
